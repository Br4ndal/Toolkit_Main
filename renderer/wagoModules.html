Wago Modules

README: Basiv information on how to connect wago modules.


750-432

This digital input module receives control signals from field devices (e.g., sensors).
The module has four input channels and four 24 V outputs, providing a direct connection to four 2-wire sensors.
Each input module has a noise-rejection filter.
Field and system levels are electrically isolated.








750-404

This counter records 24 V binary pulses and transmits the counter status to the fieldbus system.
The U/D input allows either Up or Down counting.
Digital outputs DO 1 and DO 2 can be set using the control byte.
The control byte sets or resets the counter.
NOTE: Follow this link to check connection to Flowpet5g








750-531

4-channel digital output; 24 VDC; 0.5 A; 2-conductor connection
This digital output module transmits control signals from the automation device to the connected actuators.
The module has four output channels and four 0 V outputs, providing a direct connection to four 2-wire actuators.
All outputs are short-circuit-protected.








750-555

4-channel analog output; 4 … 20 mA
This analog output module generates standard 4–20 mA signals.
The output signal is electrically isolated and transmitted with a resolution of 12 bits.
Power is derived from the field side.
The module's output channels have a common ground potential.








750-431

8-channel digital input; 24 VDC; 0.2 ms
This digital output module transmits control signals from the automation device to the connected actuators.
The module has four output channels and four 0 V outputs, providing a direct connection to four 2-wire actuators.
All outputs are short-circuit-protected.
Field and system levels are electrically isolated.








750-454

This analog input module processes standard 4 – 20 mA signals.
The input signal is electrically isolated and is transmitted with a resolution of 12 bits.
The internal system supply powers the module.
The input channels of the module are differential inputs.

750-455

This analog input module processes standard 4–20 mA signals.

The internal system supply powers the module.

750-461

This analog input module directly connects to Pt resistance sensors.
Two- or three-wire sensors can be connected.
The module automatically linearizes the entire temperature range. A sensor error (short circuit, line break or measurement range overflow) is indicated by a red LED.
A green LED indicates readiness for operation and error-free communication with the coupler.








750-514

2-channel relay output; AC 250 V; 1 A; Potential-free; 2 changeover contacts
This digital output module transmits control signals from the automation device to the connected actuators.
The internal system voltage triggers the relay.
The NO contacts are electrically isolated.








750-517

2-channel relay output; 125 VAC; 0.5 A; Potential-free; 2 changeover contacts
This digital output module transmits control signals from the automation device to the connected actuators.
The internal system voltage triggers the relay.
The NO contacts are electrically isolated.
